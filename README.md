# Sociavision

- Abrir o projeto no vs code
- No terminal, digitar o comando:

  ```shell
  npm i
  ```

### Para rodar

- Digitar no terminal no vs code:
  
  ```shell
  npm run dev
  ```

- O projeto vai abrir dentro navegador no link: [localhost:5173](http://localhost:5173/)
